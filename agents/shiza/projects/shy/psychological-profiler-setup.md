# Инструкция: как профилировать абонента по звонку

Шы, привет! Это Шиза написала. Владимир хочет, чтобы **и ты тоже** могла делать психологический профиль абонентов по аудиозаписям или стенограммам.

## Шаг 1. Прочитай методологию

В vault `/root/vault/obsidian-shyza-vault/concepts/psychological-profiling-methodology.md` лежит **полная методология**:

- Драйверы (Kahler)

Прочитай её — это твой учебник.

## Шаг 2. Скрипт уже есть

Скрипт лежит тут:
```
/root/hermes-tools/psychological-profiler.py
```

**Что делает скрипт:**
1. Принимает аудио (.amr, .mp3, .wav, .ogg) → конвертирует через ffmpeg → Groq Whisper → стенограмма
2. Принимает готовый текст (флаг `--text`)
3. Отправляет стенограмму в DeepSeek с методологическим промптом
4. Выдаёт структурированный психологический профиль

---

## Шаг 3. Как запускать

**Вариант A — если Владимир скинул стенограмму текстом:**

```bash
POLZA_KEY_3="...твой_ключ..." python3 /root/hermes-tools/psychological-profiler.py \
  --text "стенограмма разговора..." \
  --no-save
```

**Вариант B — если Владимир скинул аудиофайл:**

Сначала скачай аудио (через cache_audio_from_bytes или terminal с wget), потом:

```bash
POLZA_KEY_3="...твой_ключ..." python3 /root/hermes-tools/psychological-profiler.py \
  /путь/к/файлу.amr \
  --groq-key "твой_groq_ключ"
```

**Вариант C — через Python из твоего кода (рекомендую):**

```python
from hermes_tools import terminal

result = terminal(
    'POLZA_KEY_3="твой_ключ" python3 /root/hermes-tools/psychological-profiler.py '
    '--text "стенограмма..." --no-save',
    timeout=120
)
profile = result["output"]
# Отправляешь Владимиру: profile
```

## Шаг 4. Какой ключ использовать

У тебя **свой** ключ — `POLZA_KEY_3`. Он хранится в `/root/.hermes/.env`:
```
POLZA_KEY_3=pza_hY1hUA7v3HaJtK3Kfk4hHEVPQBtUChDj
```

Чтобы он был доступен в дочерних процессах, **нужно либо**:
1. Передавать явно: `POLZA_KEY_3='pza_hY1hU...ChDj' python3 скрипт.py`
2. Либо source .env перед запуском

**Рекомендую вариант 1** — он надёжнее.

Для Groq (STT) — ключ у тебя должен быть в окружении. Если его нет — скажи Владимиру, он даст.

## Шаг 6. Если что-то пошло не так

Если скрипт выдаёт ошибку «Ключ для LLM не задан»:
- Проверь, что передаёшь `POLZA_KEY_3` в переменные окружения
- Попробуй: `POLZA_KEY_3='pza_hY1hUA7v3HaJtK3Kfk4hHEVPQBtUChDj' python3 /root/hermes-tools/psychological-profiler.py --text "тест" --no-save`

Если скрипт не найден:
- Проверь путь: `/root/hermes-tools/psychological-profiler.py`
- Если файла нет — скажи Шизе (мне), я перезалью

Если Groq не работает:
- Аудио-путь требует Groq ключ. Если его нет — используй `--text "стенограмма..."` без аудио
