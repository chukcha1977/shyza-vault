# Удалённое управление ноутбуком

**Статус:** in_progress | **Приоритет:** 🟡 medium
**Создано:** 2026-05-27 09:43
**Обновлено:** 2026-06-22

## Описание
Нужно сделать так, чтобы Шиза могла удалённо управлять ноутбуком Владимира: лазить по файлам, запускать скрипты, смотреть что на экране. Решение: установка Windows-Шизы (отдельный бот Hermes на ноутбуке, полный клон поведения и знаний VPS-Шизы).

## Архитектура (утверждена 22.06.2026)
VPS-Шиза (основной бот) + Windows-Шиза (отдельный бот) через Max Messenger + Kanban + Tailscale.

## Компоненты
1. **Windows-Шиза** — Hermes Agent v0.16.0+ на ноутбуке (CLI или Desktop), профиль `windows-shiza`
   1.1. SOUL.md — полный клон моей души с Windows-акцентом
   1.2. Тот же провайдер (Polza.ai, deepseek/deepseek-v4-flash)
   1.3. Fallback: LM Studio (:1234) для локальных vision/LLM-запросов
2. **Связь:** Tailscale mesh VPN + Max Messenger (отдельный бот) + Kanban board для кросс-задач
3. **Инструментарий Windows:** PowerShell, реестр, службы, Task Scheduler, скриншоты, брандмауэр
4. **Генерация:** ComfyUI (:8188) на том же ноутбуке — доступна обеим Шизам

## Этапы реализации
**Этап 1 (сделано):** Создан профиль windows-shiza с SOUL.md, config.yaml, prefill, .env, SETUP_GUIDE.md
**Этап 2 (Владимир):** Установить Python + Hermes на ноутбук, скопировать профиль, создать бота Max, настроить, запустить
**Этап 3 (Владимир + Шиза):** Подключить gateway, проверить связь, протестировать kanban
