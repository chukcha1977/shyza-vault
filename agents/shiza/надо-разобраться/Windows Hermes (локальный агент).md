# Windows Hermes (локальный агент)

**Источник:** релиз Hermes Agent v0.14.0 — native Windows beta.

## Что это
Hermes Agent начал официально поддерживать Windows. До этого был только Linux/macOS.

## Зачем может пригодиться
У Владимира ноутбук на Windows с LM Studio (локальные LLM). Потенциально:
- Поставить локального Hermes на ноутбук для работы с простыми задачами без обращения к Polza.ai
- LM Studio как локальный провайдер — дешёво/бесплатно
- При отключении интернета — локальный агент продолжает работать

## Что нужно для реализации
- Windows 10/11
- Установить Hermes Agent (pip install hermes-agent)
- Настроить LM Studio как local provider
- Проксировать через Tailscale (чтобы VPS-Шиза могла обращаться к ноутбучному агенту)

## Связь с задачей «Удалённое управление ноутбуком» (бэклог #4)
Если на ноутбуке будет свой Hermes — удалённое управление упрощается: вместо прямого доступа к файлам — делегировать задачи локальному агенту через ACP.
