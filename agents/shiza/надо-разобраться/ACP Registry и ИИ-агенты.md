# ACP Registry и ИИ-агенты

**Источник:** Hermes Agent — ACP (Agent Communication Protocol) adapter встроен (уже в v0.13.0, в v0.14.0 расширен Zed/uvx интеграцией).

## Что это
Протокол, позволяющий разным ИИ-агентам общаться друг с другом:
- Агент А просит агента Б выполнить задачу
- Агент Б возвращает результат
- Встроено в Hermes: `acp_adapter/` — полноценный сервер, сессии, тулы, пермишены

## Потенциальное применение
1. **Онлайн-консультант на сайте пеноизола:**
   - Посетитель задаёт вопрос → простой ИИ-агент на сайте → если сложный вопрос → делегирует Шизе через ACP
2. **Несколько агентов в разных мессенджерах:**
   - Агент в Telegram → делегирует Шизе в Max
3. **Локальный агент на ноутбуке Владимира:**
   - Шиза на VPS → делегирует задачу «посмотри файл на диске D:/» локальному агенту на Windows
   - Тот выполняет и возвращает результат

## Что нужно для старта
- Поднять ACP сервер (`python -m acp_adapter`)
- Настроить второго агента (например, на сайте)
- Связать их по сети
