# SimpleX Chat

**Источник:** релиз Hermes Agent v0.14.0 — появилась поддержка SimpleX Chat как платформы.

## Что это
- Децентрализованный мессенджер — нет серверов, нет ID пользователей, нет телефонов/email при регистрации.
- Полная анонимность: каждый контакт — отдельный уникальный адрес (одноразовая ссылка).
- Работает через P2P и onion-роутинг.

## Зачем может пригодиться
- Анонимный канал связи для консультаций/заказов с сайта — если клиент не хочет светить номер/телеграм.
- Децентрализованное резервное общение, если Max/Telegram лягут.

## Что нужно для запуска
- Поставить приложение SimpleX на телефон/ноутбук.
- В Hermes — включить плагин платформы SimpleX Chat.
- Разобраться с одноразовыми адресами (invitation links).
