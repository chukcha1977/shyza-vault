# OpenRouter — доступ с VPS

**Суть:** OpenRouter (openrouter.ai) — агрегатор 300+ AI-моделей (Claude, Gemini, Grok, Qwen и др.) через единый API. Блокирован для РФ, но наш VPS в Германии — работает напрямую (HTTP 200, 0.5с).

**Что даёт (чего нет у нас сейчас):**
1. Claude Opus 4.8 — топ для сложного анализа и кода
2. Grok 4.3 / Grok Imagine — тексты + изображения
3. Gemini 3.5 Flash — новее нашей 3.1
4. Speech API (TTS/STT) — синтез и распознавание речи
5. Model Fusion — один запрос к 2+ моделям, синтез лучшего ответа
6. Qwen3.7 Max — гигантский контекст

**Текущий статус:** не подключён. Нужна регистрация и пополнение. Решать по необходимости — под конкретную модель, которой нет у Polza.
