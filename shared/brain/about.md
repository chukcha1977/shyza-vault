# Владимир

## Кто

Владелец бизнеса по производству **пеноизола** (черноземье — Воронеж, Липецк, Тамбов, Белгород, Курск). Производство, логистика, продажи B2B и B2C.

## Проекты

| Проект | Стек | Суть |
|--------|------|------|
| **Дашборд заказов** | FastAPI, MariaDB, Python | Учёт клиентов, заказов, себестоимости, прибыли |
| **Сайт penoizol-chernozem.ru** | 1С-Битрикс, Beget | Корпоративный сайт + SEO-контент |
| **База знаний (этот vault)** | Obsidian, GitHub, Markdown | Накопление знаний для ИИ-агента (Шизы) |

## Технологический стек

- **Пеноизол:** производство, себестоимость, карбамидные смолы
- **Веб:** FastAPI, MariaDB 10.11, Битрикс, Beget
- **ИИ:** Hermes Agent, Claude Sonnet 4 (через Polza.ai), DeepSeek V4
- **CI/Git:** GitHub, gh CLI
- **ОС:** Linux (сервер Hermes), Windows (рабочий ПК)
