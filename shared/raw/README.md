# raw/ — Layer 1: Immutable source material

По методу Карпати. Только добавление, не редактировать.

## Структура
- `raw/articles/` — Веб-статьи, исходники сайта (Joomla дамп, контент с penoizol-chernozem.ru)
- `raw/papers/` — PDF, технические документы
- `raw/assets/` — Изображения, диаграммы

## Правила
- Файлы неизменяемы (immutable). Если источник обновился — добавлять новый файл с новой датой.
- Каждый файл содержит frontmatter: source_url (если есть), ingested (дата), sha256 тела.
- **Не редактировать raw/ напрямую.** Изменения → через новый ингейст с обновлённой датой.
