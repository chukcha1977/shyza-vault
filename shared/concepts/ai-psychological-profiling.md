# AI-инструменты психологического профилирования по голосу/тексту

**Задача Владимира:** найти инструменты, которые анализируют запись телефонного разговора (или стенограмму) для определения психотипа абонента, его психологических особенностей — построение «психокарты».

**Вывод:** готового продукта «загрузил звонок → получил полную психокарту» нет. Но все компоненты существуют и собираются в пайплайн через LLM.

---

## Анализ по аудио/голосу (без текста)

### 1. Hume AI (EVI 3)
- **Что делает:** эмпатический голосовой AI, распознаёт эмоции по голосу в реальном времени (тон, тембр, интонации, стиль речи)
- **Даёт:** эмоциональный профиль спикера (гнев, тревога, радость, неуверенность и т.д.)
- **Не даёт:** Big Five / DISC напрямую
- **Есть:** open-source модели, Python SDK
- **Цена:** pay-as-you-go (точные цифры — на сайте)
- **Ссылка:** [hume.ai](https://www.hume.ai/)

### 2. Voicesense
- **Что делает:** анализ 200+ просодических параметров речи, предсказание черт личности и поведения
- **Независимость:** не зависит от языка, культуры, пола
- **Позиционирование:** колл-центры, HR, risk assessment
- **Цена:** enterprise, непублична
- **Ссылка:** [voicesense.com](https://voicesense.com/)

### 3. Inworld AI (STT Voice Profiling API)
- **Что делает:** распознаёт эмоцию, акцент, возраст, высоту тона, стиль речи + транскрипция в 1 запросе
- **Даёт:** голосовой профиль (но не глубинный психотип)
- **Цена:** pay-as-you-go
- **Ссылка:** [inworld.ai](https://inworld.ai/speech-to-text)

## Анализ по тексту/стенограмме

### 4. Sentino Personality API
- **Что делает:** NLP-движок, определяет Big Five, NEO, HEXACO, DISC по любому авторскому тексту
- **Вход:** письма, чаты, посты, стенограммы
- **Python-клиент:** есть (github.com/Yifan-Zang/sentino-api-client)
- **Цена:** custom (по запросу)
- **Ссылка:** [sentino.org](https://sentino.org/)

### 5. Receptiviti
- **Что делает:** психолингвистический анализ, 200+ метрик: Big Five, стили лидерства, эмоции, когнитивные стили
- **Наследник:** IBM Watson Personality Insights (IBM брал их технологию)
- **Цена:** API от $500/мес
- **Ссылка:** [receptiviti.com](https://www.receptiviti.com/)

### 6. Humantic AI
- **Что делает:** DISC + Big Five по email, LinkedIn, тексту
- **Интеграция:** CRM (Salesforce, HubSpot)
- **Цена:** $90-105/мес
- **Ссылка:** [humantic.ai](https://humantic.ai/)

### 7. Crystal Knows (Crystal API)
- **Что делает:** DISC-профилирование по тексту
- **Два API:** Analysis (текст → профиль) и Profile (email/LinkedIn → профиль)
- **Цена:** custom pricing
- **Ссылка:** [crystalknows.com](https://www.crystalknows.com/developers/text-analysis)

### 8. Mindreader AI
- **Что делает:** анализ коммуникативных стилей и психотипа для продаж
- **Даёт:** MBTI / Big Five по языковым паттернам
- **Ссылка:** [themindreader.ai](https://themindreader.ai/)

## Open Source / DIY

### 9. Emory NLP Personality Detection
- GitHub: emorynlp/personality-detection
- Детекция Big Five по диалогу (бинарные классы)

### 10. Speech Emotion Recognition (SER) — open-source
- Пайплайн: librosa + ML-модели
- Распознавание эмоций из аудио (без текста)

### 11. Статья MeanoTek на Хабре (2022)
- Команда построила модель на XLM-RoBERTa для Big Five из текста
- Достигли MAE 0.073
- Подтверждает, что задача решаема через fine-tune языковых моделей
- **Ссылка:** [habr.com](https://habr.com/ru/companies/meanotek/articles/689360/)

## Рекомендуемый подход (сборка из компонентов)

1. **Аудио → стенограмма:** Whisper (локально)
2. **Аудио → эмоции:** Hume AI API
3. **Стенограмма → психотип:** LLM с промптом на психотипирование (Claude/GPT/DeepSeek — Big Five, DISC, ценности, коммуникативный стиль) ИЛИ Sentino/Receptiviti API
4. **Сводка:** объединить эмоциональный профиль + текст-психотип в единую карту

Плюсы: минимальная стоимость (только Hume API), всё работает на текущих мощностях.
