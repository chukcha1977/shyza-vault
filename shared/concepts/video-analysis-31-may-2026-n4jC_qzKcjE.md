## Ключевые идеи

1. **RAG-система (vector search + синхронизация папок)** — автор кладёт PDF-конспекты в папку `мастерская`, синхронизирует через Syncthing с сервером, Hermes индексирует тексты в векторное пространство. При запросе — находит смыслово близкие фрагменты. Obsidian строит граф знаний автоматически. Три слоя: (1) папка на компе = папка на сервере (Syncthing), (2) нарезка файлов на кусочки → векторы (координаты смысла), (3) вопрос тоже вектор → поиск ближайших абзацев. Автор оценивает установку в 15-20 минут.

2. **Команда `/goal`** — агент получает цель и выполняет автономно, без вопросов, до результата. Точка А (цель) → точка Б (результат). Пример: составление PDF-справочника по 139 slash-командам Гермеса — сделал сам, без остановок. 16 страниц, структурированный результат.

3. **Команда `/ground`** — параллельное выполнение нескольких задач. Автор дал 3 команды одновременно: собрать новости про автоматизацию, проанализировать посты Telegram-канала, дать рекомендацию по следующей теме ролика. Hermes выполняет все три и выдаёт результаты по мере готовности. Бонус: сам додумался рекомендовать стиль контента (личный опыт + практический артефакт лучше, чем опробность).

4. **Codex внутри Hermes через подписку ChatGPT ($20/мес)** — автор использует Codex как встроенный инструмент Гермеса для разработки. Пример: создал HTML-коммерческое предложение в фирменном стиле через промт. Код вызывается из Hermes, скилл `cloudy` подключается автоматически. Автор подтверждает, что ChatGPT-подписка за $20 достаточно для работы Codex внутри Hermes.

5. **Память агента на SQL-базе — не тратит токены** — вместо загрузки всего контекста в промпт, Hermes ищет нужную сессию в SQL-базе, собирает выжимку с нулевой оплатой токенов. Чем больше работы с агентом, тем сильнее экономия. Пример: вспомнил скриншот с MP3-треклистом из прошлой сессии. Это ключевое улучшение — экономия прямо экономит деньги.

6. **Kanban-доска с пакетным созданием задач** — можно скинуть список из 10-20 задач в одном промпте, Hermes сам сортирует: создаёт карточки, раскидывает по колонкам, выполняет. В отличие от прошлых версий, где на каждую задачу отдельный промпт. Пример: 5 новостей про Claude, 5 идей для шортс, черновик поста — всё в одном триггере.
