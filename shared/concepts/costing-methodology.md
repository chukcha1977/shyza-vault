# Методики расчёта себестоимости

> Прочитай этот файл ПЕРВЫМ при работе в чате «шиза-бух».

## Константы (общие для обеих методик)

- Плотность смолы: **1 л = 1.25 кг**
- Цена смолы: **83 руб/кг** (повышена с 77 руб/кг с июня 2026, подтверждено расходами заказов №№32-36)
- Прочая химия (пенообразователь + кислота): **+10%** к стоимости смолы
- Выход пены: **из 100 л смолы → 6 м³ пены**
- Налог УСН: **6% с выручки** (только при безналичной оплате; наличка — налог 0)

## Выбор методики

| Если хозяин дал | Используем |
|---|---|
| Объём смолы (л) | **Методика №1** — прямой расчёт |
| Площадь стен + толщину слоя (без объёма смолы) | **Методика №2** — геометрия для плана. По факту хозяин скажет реальный расход → переходим на №1 с фактическими цифрами |

## Методика №1 — через объём смолы

```
1. Смола (кг) = объём_смолы_л × 1.25
2. Стоимость_смолы = смола_кг × 77
3. Итого_материалы = стоимость_смолы × 1.1   (смола + 10% химия)
```

## Методика №2 — через геометрию стен

```
1. Объём пены (м³) = площадь_стен_м² × толщина_слоя_м
2. Расход смолы (л) = (объём_пены_м³ × 100) / 6
3. Смола (кг) = расход_смолы_л × 1.25
4. Стоимость_смолы = смола_кг × 77   (внимание: здесь 77, не 72!)
5. Итого_материалы = стоимость_смолы × 1.1
```

## Себестоимость и прибыль

```
Себестоимость = материалы + ЗП + транспорт + прочее_по_заказу + налог_УСН
Выручка = сумма_заказа
Налог УСН 6% = выручка × 0.06   (только для безнала)
Чистая прибыль = выручка - себестоимость
Рентабельность = (чистая_прибыль / себестоимость) × 100%
```

## Важные правила

- **Маркеры (план)/(факт):** каждый expense description ОБЯЗАН заканчиваться на `(план)` или `(факт)`. Без маркера дашборд ломает расчёт (случай Кондратьева).
- **Доставка смолы:** прямой расход на заказ (`order_id = #N`), не overhead.
- **Знак расходов:** все отрицательные.
- **Вычет окон/дверей:** если не сказано «чистая площадь» — вычесть 10% (×0.9).
- **Способ оплаты:** по умолчанию `cash`. `noncash` → начисляется УСН 6%.
- **Топливо:** км × 3 руб/км. Если не назван — не добавлять.
- **Не озвучил = плановое:** если при вводе факта параметр не назван — берётся плановое значение.
- **Накладные (overhead):** НЕ входят в себестоимость заказа. Только прямые расходы с `order_id`.
