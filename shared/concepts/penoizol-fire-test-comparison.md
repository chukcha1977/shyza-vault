# Испытание на горючесть: пеноизол vs 9 материалов

## Определение

Лабор��торный эксперимент (Беларусь): 10 теплоизоляционных материалов протестированы контролируемым возгоранием с бензином. Замерялись: продолжительность горения, температура, дымность, самозатухание, характер повреждений.

## Ключевые выводы

### Пеноизол — лучший по пожаробезопасности
- **Не горел самостоятельно** — горение только там, где был бензин
- **Минимальная дымность** — безопасен при пожаре для дыхания
- **Практически без потери объёма** — сохраняет теплоизоляцию после пожара
- Единственный минус: неприятный запах при горении

### Худшие результаты
- **Брендовый ЭППС (Россия)** — 280 с горения, 100% сгорел, бурная химреакция с бензином. Абсолютный антилидер
- **Минвата** — 121 с горения (впитывает горючее), самая высокая дымность
- **ППУ** — температура >400°С, едкий удушливый дым

### Сравнение рыночных конкурентов
- Пе��ополистиролы (ППС, ЭППС) — горят, оплавляются, теряют объём, не самозатухают
- Пеноизол — не поддерживает горение, не теряет объём
- Пеноизол = класс Г1 (слабогорючий), ППС/ЭППС = Г3-Г4, ППУ = Г3

## Почему это важно

1. Единственный в тесте материал, горевший ТОЛЬКО за счёт горючего — после выгорания бензина горение прекращалось
2. Не выделяет токсичного дыма (в отличие от ППУ и минваты)
3. Сохраняет теплоизоляционные свойства после возгорания (нет потери объёма)
4. Эти данные — сильный аргумент для сайта: уникальный эксперимент, не встречается у конкурентов
