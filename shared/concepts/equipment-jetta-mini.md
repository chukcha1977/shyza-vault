# Оборудование Джетта-Мини — поколения и сравнение

## Определение

Классификация и сравнение трёх поколений (систем) оборудования для производства жидкого пеноизола (карбамидного пенопласта) на объекте. Описание установки «Джетта-Мини» — последнего поколения турбомеханических систем.

## Характеристики установки «Джетта-Мини»

| Параметр | Значение |
|----------|----------|
| Производительность | 4–12 м³/ч |
| Плотность пеноизола | 6–50 кг/м³ |
| Потребляемая мощность | 0,2 кВт (установка) + 1,6 кВт (компрессор) — всего 1,8 кВт |
| Напряжение | 220 В |
| Масса | 10 кг |
| Длина подающего рукава | 25/33/50 м (стандарт 25 м) |
| Диаметр сужающих насадок | 10–20 мм |
| Система «Старт-Стоп» | ✅ есть |
| Система «Работа зимой» | ✅ есть |
| Регулировка плотности | по 3 каналам |
| Обслуживающий персонал | 1 человек |
| Гарантия | 12 мес. (пульт) / 36 мес. (насосы) |
| Подача на высоту | до 50 м |

## Три поколения оборудования

| Характеристика | ГЖУ (1-е поколение) | «Поток» (2-е поколение) | Джетта-Мини (3-е поколение) |
|----------------|---------------------|------------------------|---------------------------|
| Обслуживание | 2-3 чел. | 2-3 чел. | **1 чел.** |
| Качество | неоднородное, нестабильное | однородное, высокое | однородное, высокое |
| Старт-Стоп | ❌ | ❌ | **✅** |
| Промывка шлангов | обязательна | обязательна | **не нужна** |
| Подг./заключ. время | ~2 часа | ~1 час | **5 минут** |
| Диаметр отверстий | >40 мм | 18-30 мм | **10-18 мм** |
| Регулировка | практически нет | нет | **по 3 каналам** |

| Многоэтажки | ❌ | ❌ | **✅** |

## Технологический процесс

Три ключевых процесса в каждом поколении выполняются в разной последовательности:

- **ГЖУ**: Пеногенерация (1) → Инжекция смолы (2) → Смешивание в шланге (3)
- **Поток**: Смешивание+Инжекция (1) → Пеногенерация в шланге (2)
- **Джетта-Мини**: Все три процесса одновременно в турбине — вспенивание, смешивание и инжекция в одной точке

## Ключевые выводы

1. **«Старт-Стоп»** — революционная функция: остановка заливки без промывки шлангов. Экономия сырья (150-500 руб. на каждой промывке) и времени
2. **Совместимость поколений**: насосы Джетта-Мини взаимозаменяемы с «Поток» — можно модернизировать старую установку, заменив смесительную систему
3. **Низкий порог входа**: 1 оператор, 5 мин подготовка, 10 кг вес — можно утеплять любой дом за 1 день
4. **Для сайта**: уникальное УТП — «утепляем многоэтажки, работаем зимой, один мастер, без промывок»
5. **Окупаемость оборудования**: ~50 м³ заливки (≈120 000 руб.) — больше смысла купить, меньше — нанять подрядчика
