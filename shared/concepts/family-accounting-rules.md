# Семейная бухгалтерия — правила работы

> Прочитай этот файл при работе в чате «Семейный учет».

## 🔥 Идентификация отправителя — каждое сообщение

Никогда не кэшировать отправителя. Каждое сообщение — отдельная проверка.

**ШАГ 0 для любого сообщения в семейном чате:**
```bash
/root/.hermes/scripts/get_sender.sh
```
→ возвращает `Владимир` или `Саша` или `unknown`.

Не доверять системной подсказке User — она фиксирована на сессию.
Не гадать. Всегда запускать скрипт.

## Три сценария

### А — Фото чека
1. Определить отправителя (скрипт)
2. Vision → распознать: магазин, сумма, дата, товары
3. Показать хозяину кратко (4-5 строк):
   `🧾 [Владимир/Саша] Магазин | Сумма ₽`
4. Ждать «да» → записать через receipt_to_db.py
5. НЕ додумывать содержимое. Если не уверена — попросить продиктовать.
6. Количество чеков на картинке = количество показываемых. Не додумывать из контекста.

### Б — Ручной ввод
1. Определить отправителя (скрипт)
2. Показать одной строкой:
   `Записать: Категория — Сумма ₽ (описание)?`
3. Ждать «да» → записать

### В — Отчёт
SQL к chuchm_finance. Строгая нумерация. Эмодзи-шапка.

## Важные правила
- Ничего не писать без подтверждения хозяина
- Расходы — отрицательные в БД (receipt_to_db.py сам ставит знак)
- Все расходы по умолчанию «Прочее», если хозяин не уточнил категорию
- Чек смешанный (продукты + бытовая химия) — спросить, разбивать ли
- При сомнениях в распознавании — честно сказать «не уверена, продиктуй»
